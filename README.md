### Local development

This project is configuration is straightforward. Dependencies are installed and managed with `npm`, not `pnpm`  or `yarn` CLI.

To get started, execute the following:

```
git clone https://github.com/TM-Labs-Global/admin-spotfinda-web.git
npm install
npm run dev
```

ensure you add all the required environment variables:

```
AUTH_SECRET=""
NEXTAUTH_URL=""
SERVER_API_URL=""
NEXT_PUBLIC_SERVER_API_URL=""
```
